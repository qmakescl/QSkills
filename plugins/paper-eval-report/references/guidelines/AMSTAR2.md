# AMSTAR 2 — SR 방법론적 질 평가 도구

- **발표:** 2017 (항목 수: 16개, Critical domain 7개)
- **대상:** 무작위/비무작위 중재 연구를 포함한 체계적 문헌고찰 (진단/Scoping/NMA 제외)

---

## 16개 체크리스트 (★ = Critical)

| 번호 | 항목 내용 | Critical |
|------|-----------|---------|
| 1 | 연구 질문의 PICO 구성 요소(P, I, C, O)를 포함하는 사전 프로토콜 작성 여부 | |
| **2** | **문헌 선택 및 데이터 추출이 독립적·이중으로 수행되었는지** | **★** |
| 3 | 포괄적 문헌 검색 전략 수행 여부 | |
| 4 | 회색 문헌(grey literature) 검색을 포함한 출판 유형에 제한 없는 검색 여부 | |
| **5** | **제외 연구 목록 및 제외 이유 제시 여부** | **★** |
| 6 | 포함 연구의 특성 기술 여부 | |
| 7 | 포함 연구의 재원 출처 보고 및 그 영향 논의 여부 | |
| **8** | **RCT에 대한 비뚤림 위험 평가에 적절한 도구 사용 여부** | **★** |
| **9** | **비무작위 연구(NRSI)에 대한 비뚤림 위험 평가에 적절한 도구 사용 여부** | **★** |
| 10 | 메타분석에 포함된 연구들의 통계적 이질성(heterogeneity) 보고 여부 | |
| **11** | **메타분석 수행 시 이질성에 따른 적절한 통계적 방법 사용 여부** | **★** |
| 12 | 개별 연구의 비뚤림 위험이 결과 해석에 미치는 영향 논의 여부 | |
| **13** | **개별 연구의 비뚤림 위험을 고려한 결과 해석 여부** | **★** |
| 14 | 출판 비뚤림(publication bias) 조사 및 그 영향 논의 여부 | |
| **15** | **이해충돌이 결론에 미치는 잠재적 영향 보고 및 논의 여부** | **★** |
| 16 | 포함 연구의 비뚤림 위험을 설명하는 이질성 원인 탐색 여부 | |

★ = **Critical domain** (7개)
