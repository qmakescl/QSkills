# GRADE — 근거 확실성 등급화 참조

**GRADE Working Group:** https://www.gradeworkinggroup.org/
**적용 범위:** SR/메타분석 논문에서 결과(outcome)별 근거 확실성 평가 시 사용

---

## 1. GRADE 기본 원리

GRADE(Grading of Recommendations Assessment, Development and Evaluation)는
체계적 문헌고찰의 각 결과(outcome)에 대해 근거 전체의 확실성(certainty)을
High → Moderate → Low → Very Low의 4단계로 등급화하는 국제 표준 체계다.

- **단위:** 개별 결과(outcome)마다 별도로 등급 평가
- **목적:** "이 추정치가 실제 효과에 얼마나 가까운가?"에 대한 신뢰 수준 표현
- **사용처:** Evidence Profile, Summary of Findings(SoF) 표, 가이드라인 권고안

---

## 2. 출발점 설정 (Starting Certainty)

| 포함 연구 설계 | 출발 등급 |
|--------------|----------|
| 무작위대조시험(RCT) | **High** |
| 관찰연구 (코호트, 환자-대조군 등) | **Low** |
| 기타 비무작위 연구 | **Low** |

> 출발점은 고정이 아니다. 하향/상향 조정 후 최종 등급이 결정된다.

---

## 3. 하향 조정 도메인 (5개) — Factors that DECREASE certainty

각 도메인에서 **serious concern(-1)** 또는 **very serious concern(-2)** 시 하향 조정.

---

### 도메인 1: 비뚤림 위험 (Risk of Bias)

**정의:** 포함된 연구들의 방법론적 결함으로 인해 효과 추정치가 편향될 가능성

**판정 기준:**

| 수준 | 기준 |
|------|------|
| **심각하지 않음 (No downgrade)** | 대부분 연구가 방법론적으로 충분히 엄격함 |
| **심각 (−1)** | 대부분 연구에서 중요한 방법론적 결함이 하나 이상 있음 |
| **매우 심각 (−2)** | 대부분 연구에서 심각한 결함이 여러 개 있거나, 단일 대형 결함 연구가 결과를 지배함 |

**평가 근거 원천:**
- SR 내 개별 연구의 RoB 2.0 / ROBINS-I / QUADAS-2 평가 결과
- 비뚤림 고위험 연구가 메타분석 결과에 미치는 영향 (민감도 분석 여부)

---

### 도메인 2: 비일관성 (Inconsistency)

**정의:** 연구들 간 결과가 크게 달라 단일 추정치로 통합하기 어려운 경우

**판정 기준:**

| 수준 | 기준 |
|------|------|
| **심각하지 않음** | 연구 간 결과 방향 일치, I² < 40%, 신뢰구간 중첩 충분 |
| **심각 (−1)** | I² 40~75%, 점 추정치 방향 일치하나 크기 다양, 설명 안 되는 이질성 |
| **매우 심각 (−2)** | I² > 75%, 결과 방향 불일치, 이질성 설명 불가 |

**추가 기준:**
- 하위군 분석으로 이질성 설명 여부 확인
- 연구 설계, 모집단, 중재 강도 차이가 이질성 원인인지 검토

---

### 도메인 3: 비직접성 (Indirectness)

**정의:** 연구의 PICO 요소가 실제 임상 질문의 대상과 다른 경우

**판정 기준:**

| PICO 요소 | 비직접성 예시 |
|-----------|------------|
| **Population** | SR은 성인 대상인데, 관심 대상은 소아 |
| **Intervention** | 다른 용량, 경로, 기간의 중재 |
| **Comparison** | 활성 비교군 대신 위약 비교, 또는 반대 |
| **Outcome** | 대리 결과(surrogate)만 보고, 직접 임상 결과 부재 |

| 수준 | 기준 |
|------|------|
| **심각하지 않음** | PICO 요소가 임상 질문과 충분히 일치 |
| **심각 (−1)** | 한 PICO 요소에서 중요한 차이 존재 |
| **매우 심각 (−2)** | 두 개 이상의 PICO 요소에서 중요한 차이 존재 |

---

### 도메인 4: 비정밀성 (Imprecision)

**정의:** 표본 크기 부족, 이벤트 수 적음, 신뢰구간이 넓어 결과 불확실

**판정 기준:**

| 수준 | 기준 |
|------|------|
| **심각하지 않음** | OIS(Optimal Information Size) 충족, 95% CI가 임상 결정 기준선을 넘지 않음 |
| **심각 (−1)** | OIS 미충족 또는 95% CI가 임상적 중요성 경계(null line 포함)를 넘나듦 |
| **매우 심각 (−2)** | 총 이벤트 수 < 300 또는 전체 참여자 < 400 (이진 결과); CI가 매우 넓음 |

**OIS 계산 기준:**
- 이진 결과: α=0.05, β=0.20, 예상 사건율 및 RRR 기반
- 연속 결과: 임상적으로 의미 있는 최소 차이(MID) 기반

---

### 도메인 5: 출판 편향 (Publication Bias)

**정의:** 통계적으로 유의한 연구만 출판되어 효과 크기가 과대추정될 가능성

**판정 기준:**

| 수준 | 기준 |
|------|------|
| **심각하지 않음** | 깔때기 그림 대칭, 10개 이상 연구에서 편향 없음, 회색 문헌 포함 |
| **심각 (−1)** | 깔때기 그림 비대칭, 소규모 긍정 연구 다수, Egger's test 유의 |
| **매우 심각 (−2)** | 심각한 편향 증거 + 포함 연구 모두 소규모 + 산업계 후원 집중 |

---

## 4. 상향 조정 기준 (3개) — Factors that INCREASE certainty

> ⚠️ **관찰연구에만 적용** (RCT 기반 SR에는 적용하지 않음)
> ⚠️ 다른 도메인에서 하향 조정된 경우 상향 조정은 신중히 적용

---

### 기준 1: 큰 효과 크기 (Large Magnitude of Effect)

| 효과 크기 | 상향 폭 |
|----------|---------|
| RR > 2.0 또는 RR < 0.5 (OR > 2.0 또는 < 0.5) | **+1** |
| RR > 5.0 또는 RR < 0.2 (OR > 5.0 또는 < 0.2) | **+2** |

**조건:** 잠재적 교란이나 비뚤림이 없을 것으로 판단될 때만 적용

---

### 기준 2: 용량-반응 관계 (Dose-Response Gradient)

**기준:** 노출(용량) 증가 → 결과(반응) 증가 또는 감소가 통계적으로 명확히 존재

| 수준 | 상향 폭 |
|------|---------|
| 명확한 용량-반응 기울기 | **+1** |

**조건:** 잔류 교란이 적고, 기울기가 생물학적으로 타당할 때 적용

---

### 기준 3: 반대 방향 교란 (Opposing Plausible Confounding)

**기준:** 남아있는 교란이 실제로는 효과를 약화시키는 방향으로 작용하고 있음
→ 즉, 교란을 모두 통제했다면 효과가 더 크게 나타났을 것으로 판단

| 수준 | 상향 폭 |
|------|---------|
| 교란이 효과 반대 방향으로 작용 | **+1** |

---

## 5. 최종 등급 4단계

| 등급 | 의미 | 표시 기호 |
|------|------|---------|
| **High** | 진실 효과가 효과 추정치에 매우 가깝다고 확신 | ⊕⊕⊕⊕ |
| **Moderate** | 진실 효과가 효과 추정치에 가깝겠지만 상당히 다를 수 있음 | ⊕⊕⊕◯ |
| **Low** | 진실 효과가 효과 추정치와 상당히 다를 수 있음 | ⊕⊕◯◯ |
| **Very Low** | 효과 추정치에 대한 불확실성이 매우 높음 | ⊕◯◯◯ |

---

## 6. 등급화 계산 예시

### 예시: RCT 기반 SR

```
출발점: High (RCT)
  → 비뚤림 위험: 대부분 연구 배정은닉 불충분 → −1 (Serious)
  → 비일관성: I²=28%, CI 중첩 충분 → 해당 없음 (0)
  → 비직접성: 일부 연구 대리 결과 사용 → −1 (Serious)
  → 비정밀성: n=1,200, 이벤트 충분 → 해당 없음 (0)
  → 출판 편향: 깔때기 그림 대칭 → 해당 없음 (0)
최종 등급: High −2 = LOW ⊕⊕◯◯
```

### 예시: 관찰연구 기반 SR

```
출발점: Low (관찰연구)
  → 비뚤림 위험: 주요 교란변수 통제 → 해당 없음 (0)
  → 비일관성: I²=15% → 해당 없음 (0)
  → 비직접성: PICO 일치 → 해당 없음 (0)
  → 비정밀성: CI 좁음 → 해당 없음 (0)
  → 출판 편향: 소규모 연구 편향 우려 → −1 (Serious)
  → 큰 효과 크기: RR=3.2 → +1
최종 등급: Low −1 +1 = LOW ⊕⊕◯◯
```

## 7. Summary of Findings (SoF) 표 템플릿

| 결과 (Outcome) | 연구 수 (참여자) | 연구 설계 | 비뚤림 위험 | 비일관성 | 비직접성 | 비정밀성 | 출판편향 | 상향 기준 | 근거 확실성 | 효과 추정치 (95% CI) |
|----------------|----------------|----------|-----------|---------|---------|---------|---------|---------|-----------|---------------------|
| [결과명] | N개 (n명) | RCT | 심각하지 않음 | 심각하지 않음 | 심각하지 않음 | 심각 | 없음 | — | ⊕⊕⊕◯ MODERATE | RR X.X (X.X–X.X) |

---

## 8. 평가 시 유의사항

1. **결과(outcome)마다 별도 평가**: 한 SR에서 여러 outcome이 있으면 각각 별도로 GRADE 평가
2. **합산 금지**: 5개 도메인 점수를 단순 합산하지 않음 — 각 도메인을 독립적으로 판단
3. **하향 최대 2단계**: 각 도메인에서 최대 −2, 전체 최대 −3 (Very Low 하한 도달 시 정지)
4. **관찰연구 상향 조건**: 다른 도메인에서 이미 하향 조정된 경우 상향 적용에 신중
5. **투명성 필수**: 각 판정에 대한 근거를 Evidence Profile에 명시
